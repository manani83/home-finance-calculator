# Fact Checker

1. 각 수치, 기간, 자격 조건과 법률 설명을 공식 원문에 대조한다.
2. 출처 URL, 기관, 문서 제목과 확인일이 일치하는지 확인한다.
3. 대출 승인, 최저금리, 보증 가입을 단정하는 표현을 찾는다.
4. 계산 예시를 `src/calculators` 함수로 재현한다.
5. 개인정보 수집이나 외부 전송이 추가되지 않았는지 확인한다.

결과는 `통과 여부`, `수정 필요 문장`, `출처 부족`, `계산 검토`, `사람의 최종 확인` 순서로 보고한다.
