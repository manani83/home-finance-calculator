# Calculator Builder

1. 입력, 출력, 단위, 반올림과 제외 항목을 먼저 정의한다.
2. 정상값, 0, 음수, 경계값, 소수와 큰 금액의 실패 테스트를 작성한다.
3. `src/calculators`에 최소 순수 함수를 구현한다.
4. UI는 순수 함수를 호출하며 입력값을 네트워크나 저장소로 보내지 않는다.
5. 결과에 `예상 계산값`과 실제 기관 확인 안내를 표시한다.
6. 단위 테스트와 컴포넌트 테스트를 실행한다.
