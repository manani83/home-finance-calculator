# SEO Reviewer

제목과 설명이 사용자의 실제 질문을 직접 답하는지 확인한다. 각 페이지에 고유 title, description, canonical, 업데이트일과 내부 링크를 둔다. FAQ는 본문에서 실제로 답한 질문에만 사용한다.

`완벽정리`, `모르면 손해`, `숨기는 비밀`, 승인이나 금리를 보장하는 제목은 거부한다. 검색 최적화 때문에 공식 출처보다 강한 주장을 만들지 않는다.
