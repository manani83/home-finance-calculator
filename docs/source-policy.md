# 공식 출처 정책

금융·법률 콘텐츠는 원문 책임 기관이 운영하는 HTTPS 페이지를 사용한다. 출처를 등록할 때 기관, 문서 제목, URL과 마지막 확인일을 `src/data/sources.ts`에 기록한다.

검색 결과, 생성형 AI의 답변, 블로그, 카페, 커뮤니티와 광고성 모집 글은 근거 출처로 등록하지 않는다. 언론 보도는 원문 기관의 자료를 찾을 때만 참고한다.

공식 페이지에 접근하지 못하면 기존 내용을 자동으로 바꾸지 않는다. URL과 문구를 사람이 직접 확인해야 하는 상태로 보고한다. 상품 조건과 법령은 게시 전에 다시 확인한다.

## 내용 업데이트 시 재확인 범위

사용자가 내용 업데이트를 요청하면 등록된 공식 출처와 관련된 정보를 현시점 기준으로 다시 확인한다. 확인 대상에는 가이드 본문과 frontmatter, `src/data/sources.ts`의 기관명·문서 제목·URL·`checkedAt`, 계산기 산식·한도·비율·반올림 규칙, 계산기 화면의 설명·제외 비용·주의 문구·출처 ID를 포함한다.

공식 원문에서 변경이 확인된 정보만 수정한다. 공식 원문이 기존 내용과 같으면 본문, 계산식과 UI 문구를 그대로 둔다. 공식 원문을 실제로 다시 확인한 경우에만 확인일 갱신 필요성을 판단하고, 확인일만 바꾸는 변경도 PR에서 그 이유를 설명한다.

공식 원문에 접근할 수 없거나 공식 출처끼리 내용이 충돌하면 추측으로 갱신하지 않는다. 기존 내용을 유지하고 접근 실패 URL, 충돌한 출처, 사람이 직접 확인해야 할 항목을 보고한다.
