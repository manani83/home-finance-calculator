# Calculator Explanations Design

## Goal

세 계산기 페이지에 계산 결과를 이해하고 검증하는 데 필요한 산식, 예시, 반올림 규칙, 미포함 항목, 주의사항과 공식 출처를 제공한다.

## Design

- 계산 로직과 입력 폼은 변경하지 않는다.
- `CalculatorExplanation` 공통 컴포넌트가 동일한 정보 구조를 렌더링한다.
- 계산기별 콘텐츠는 계산기 컴포넌트 가까이에 명시적인 데이터로 둔다.
- 공식 출처는 `src/data/sources.ts`에 등록된 ID만 참조한다.
- 정보 섹션은 계산 결과 유무와 관계없이 계산기 아래에 노출한다.
- 금액 예시는 현재 순수 함수의 산식과 원 단위 반올림 규칙을 그대로 설명한다.

## Required Sections

1. 계산 방법
2. 계산 예시
3. 반올림 규칙
4. 계산에 포함되지 않는 항목
5. 사용 전 확인사항
6. 공식 출처

## Testing

- 공통 컴포넌트가 모든 필수 제목과 공식 출처 링크를 렌더링하는지 검증한다.
- 각 계산기 컴포넌트가 해당 계산기의 산식과 대표 예시를 노출하는지 검증한다.
- 기존 계산 동작 테스트를 유지한다.
