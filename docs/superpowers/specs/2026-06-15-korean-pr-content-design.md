# Korean Pull Request Content Design

## Goal

Codex가 생성하거나 수정하는 PR의 사람이 읽는 내용을 모두 한글로 작성한다.

## Scope

- PR 제목을 한글로 작성한다.
- 본문의 섹션 제목, 변경 요약, 검증 결과, 금융 검토와 사람의 확인 요청을 한글로 작성한다.
- 명령어, 파일 경로, 브랜치명, 코드 식별자와 제품 고유명사는 정확성을 위해 원문 표기를 유지할 수 있다.
- 이미 열린 PR에 새 규칙을 적용할 필요가 있으면 제목과 본문을 한글로 수정한다.

## Safety

기존 브랜치 생성, 검증, 커밋, 푸시, PR 생성 순서와 자동 병합·운영 배포 금지 원칙은 변경하지 않는다.

## Verification

`AGENTS.md`에 한글 작성 대상과 예외가 명확히 적혀 있는지 확인한다. 현재 PR의 제목과 본문을 수정한 뒤 GitHub에서 결과를 다시 조회한다.
