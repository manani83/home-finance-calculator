# Typography Hierarchy Design

## Selected Direction

시안 B 균형형을 적용한다. 기존 녹색 브랜드와 밝은 배경은 유지하면서 제목, 도입문, 섹션 제목과 본문의 명도·크기·굵기 차이를 확대한다.

## Visual Rules

- 페이지 주제목은 `--heading-strong` 색상과 800 이상의 굵기를 사용한다.
- 제목 직후 도입문은 `--text-secondary` 색상으로 낮추되 본문보다 약간 크게 유지한다.
- 계산기 설명 영역은 3px 구분선으로 새 콘텐츠 시작점을 명확히 한다.
- 설명 카드에는 연한 녹색 표면과 녹색 소제목을 사용한다.
- 정책·가이드의 H2는 본문보다 충분히 크고 짙게 표시한다.
- 본문은 너무 옅어지지 않도록 `--text-body`를 사용한다.
- 모바일에서도 제목이 과도하게 커지거나 가로로 넘치지 않아야 한다.

## Scope

`src/app/globals.css`의 색상 토큰과 타이포그래피 스타일만 변경한다. 콘텐츠, 계산 로직, 광고 및 개인정보 동작은 변경하지 않는다.
