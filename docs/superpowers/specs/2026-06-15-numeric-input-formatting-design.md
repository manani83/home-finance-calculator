# Numeric Input Formatting Design

## Goal

모든 계산기 숫자 입력란에 입력 즉시 3자리 쉼표를 표시하고, 금액과 금리 입력란의 크기를 동일하게 유지한다.

## Scope

- `MoneyInput`은 정수 금액의 3자리 쉼표를 표시한다.
- `RateInput`은 소수점을 유지하면서 정수부에 3자리 쉼표를 표시한다.
- 계산기 상태와 계산 함수에는 쉼표가 제거된 숫자 문자열만 전달한다.
- 음수, 삭제, 소수점 입력과 붙여넣기를 지원한다.
- 금융 계산식과 결과 반올림 규칙은 변경하지 않는다.

## Design

`src/components/calculators/money-input.tsx`에 숫자 문자열의 표시값과 원본값을 변환하는 순수 함수를 둔다. 입력 요소는 쉼표를 표시할 수 있도록 `type="text"`를 사용하고, 금액은 `inputMode="numeric"`, 금리는 `inputMode="decimal"`을 유지한다. 허용하지 않는 문자는 제거하며 금리는 첫 번째 소수점만 보존한다.

레이아웃은 공통 CSS에서 `.field`, `.input-with-unit`, 입력 요소에 `min-width: 0`과 전체 너비를 명시한다. 두 열 grid 안에서도 금액과 금리 입력이 같은 열 너비를 넘지 않게 한다.

## Testing

- 금액 입력 `1000000`이 `1,000,000`으로 표시되는지 검증한다.
- 금리 입력 `1234.56`이 `1,234.56`으로 표시되는지 검증한다.
- 쉼표가 표시된 상태에서도 기존 계산 결과가 유지되는지 검증한다.
- 입력 요소가 공통 텍스트 타입과 모바일 키패드 힌트를 사용하는지 검증한다.
- 필수 전체 검증 명령을 실행한다.
