# 게시 체크리스트

- [ ] 모든 사실 주장에 등록된 공식 출처가 연결되어 있다.
- [ ] 공식 페이지를 현재 날짜에 직접 확인했다.
- [ ] 내용 업데이트 요청인 경우 가이드, 출처 메타데이터, 계산식, 한도, 비율, 반올림 규칙과 UI 설명을 현시점 공식 원문에 대조했다.
- [ ] 공식 원문에서 변경이 확인된 항목만 수정했고, 변경이 없는 본문·계산식·UI 문구는 그대로 두었다.
- [ ] `updatedAt`과 출처 `checkedAt`을 갱신했다.
- [ ] 확정적 금융 표현과 상품 가입 유도 표현이 없다.
- [ ] 계산 예시가 순수 함수의 실제 결과와 일치한다.
- [ ] 계산식, 한도, 비율 또는 반올림 규칙을 변경한 경우 관련 순수 함수와 테스트, UI 설명과 가이드를 함께 갱신했다.
- [ ] 개인정보 또는 금융정보를 수집하지 않는다.
- [ ] 공통 금융 면책과 필요한 법률 면책이 표시된다.
- [ ] `npm run validate:content`와 전체 검증 명령이 통과한다.
- [ ] Netlify Deploy Preview에서 모바일·데스크톱 화면과 외부 링크를 확인했다.
- [ ] 사람이 출처 원문과 변경 내용을 검토했다.
